# Белот резултат

Самостоятелна статична страница за записване на резултати при игра на белот.

## Как работи

- Чист `HTML/CSS/JS`, без build стъпка.
- Основният файл е `index.html`.
- Данните се пазят в `localStorage` на устройството.
- Пази текущата игра, раздаванията и броя спечелени игри.
- Има custom keypad, за да не се отваря телефонната клавиатура.

## Deploy във Vercel

1. Add New... -> Project
2. Избери GitHub repo `belot`
3. Framework: `Other`
4. Build command: празно
5. Output directory: празно или `.`
6. Deploy
